FEAR BENEATH IS AN INDIE HORROR GAME BASED ON 4 FRIENDS WHO GO ON A CAMP IN THE FOREST NOT KNOWING WHAT WAITS FOR THEM IN IT
THIS GAME CONTAINS HORROR SCENES, LOUD SOUNDS, PANIC MOMENTS AND MUCH MORE
THE STORY OF THE GAME IS INTENSE AND FULL OF MYSTERY
HOW TO PLAY--
W A S D to move 
SHIFT to sprint
T to turn the flashlight ON/OFF
RIGHT CLICK to skip to the next dialogues for ryan (in game character) in the game
E to interact with things in the game
